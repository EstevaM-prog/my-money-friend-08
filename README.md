# My Money Friend

## Sobre o Projeto

O My Money Friend é uma aplicação moderna e intuitiva para controle de finanças pessoais. Com ele, você pode gerenciar suas receitas, despesas, definir metas financeiras e visualizar relatórios detalhados da sua saúde financeira.

## Tecnologias Utilizadas

- **Vite** + **React**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Lucide React** (ícones)
- **Recharts** (gráficos)

## Como rodar o projeto localmente

1. Clone o repositório
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Funcionalidades

- Dashboard completo com resumo mensal
- Lançamento de receitas e despesas
- Gerenciamento de múltiplas contas e cartões
- Acompanhamento de metas financeiras
- Relatórios detalhados com gráficos
- Calendário financeiro
- Tema claro e escuro
